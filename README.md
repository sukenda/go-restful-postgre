# Go Restful Api

Go restful api build for learning go using fiber and mongodb

# API Spec

## Register account

Request :

- Method : POST
- Endpoint : `/api/signup`
- Header :
    - Content-Type: application/json
    - Accept: application/json
- Body :

```json 
{
  "username": "string",
  "password": "string",
  "email": "string",
  "phone": "string"
}
```

Response :

```json 
{
  "code": "int",
  "status": "string",
  "data": {
    "id": "string",
    "username": "string",
    "password": "string",
    "email": "string",
    "phone": "string"
  }
}
```

## Login

Request :

- Method : POST
- Endpoint : `/api/login`
- Header :
    - Content-Type: application/json
    - Accept: application/json
- Body :

```json 
{
  "username": "string",
  "password": "string"
}
```

Response :

```json 
{
  "code": "int",
  "status": "string",
  "data": {
    "access_token": "string",
    "user": {
      "id": "string",
      "username": "string",
      "email": "string",
      "phone": "string"
    }
  }
}
```
